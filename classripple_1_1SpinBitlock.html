Detailed Description

A specialized 16-way spinlock used to protect inner node branches.

This class packs 16 separate spinlocks into a single 16-bit value. It makes it possible to lock any one lock at once or, alternatively, all together.

The implementation tries to use portable constructs but has to be low-level for performance.
